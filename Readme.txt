This project is developed with Django 2.2 framework. It is a clone version of producthunt actual website.

User can:
    Signup
    Login
    Create product
    Upvote product
    delete product
    Detail product

Highlights:
    Use of images(to store images attached by user in the form)  
    User authentication
    Use of static files
    How to allow only authenticated user(already login) see some specific details

Backend used for this project is PostgreSQL.

Note:
    '*' means one needs to provide their own password/key in te respective fields.
    This project is done in virtual environment. It's highly recommended to use virtual environment for this project.
